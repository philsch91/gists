# zellij

## zellij options
```
# manage zellij sessions
zellij ls|list-sessions
zellij attach <session-name>
zellij --debug attach --create foo
#
zellij --version
zellij --config <file>
#
echo $ZELLIJ_CONFIG_DIR
```

## zellij key bindings
```
Ctrl+q # quit
```

## zellij files
